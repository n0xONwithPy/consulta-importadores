Como Atualizar os Dados do meu Importador?

Para atualizar os dados do importador o processo é relativamente simples, você precisa salvar seu arquivo do cliente por cima do arquivo Base dentro da pasta do seu Importador, feito isso, acesse o Importador e clique no botão "Atualizar Dados".

Como Gerar meus arquivos para Importação?

Para gerar seus arquivos você primeiro precisa atualizar os dados do seu importador, após se certificar que os dados foram atualizados com sucesso, clique no botão Gerar CSV, seus arquivos estarão dentro da pasta "Arquivos para Importação" prontos para serem importados, leve-os até seu G onde você deve criar uma pasta especialmente para isso. (Lembre de conferir antes de importar)

Como Importar meus Arquivos?

Para Importar os arquivos após conferi-los, Acesse o Dominio, selecione a empresa que deseja realizar a importação, clique na Guia Utilitários, Importação, Importador, Importar, no campo Conjunto de Dados selecione "Lançamentos em Lote arquivo CSV - PADRAO REAL (Lançamentos Arquivos CSV - PADRAO REAL), na caixa ao lado de "Caminho" localize seus arquivos para Importação e clique em Importar.
